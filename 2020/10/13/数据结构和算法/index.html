数据结构和算法

  1. 时间复杂度(对一个算法运行时间长短的量度)

    • 基本操作执行次数（ T(n) = 3n, T(n) = 5logn, T(n) = 2, T(n) = 0.5n 2 + 0.5n ）

    • 渐进时间复杂度

      • 如何推导出时间复杂度
        • 如果运行时间是常数量级，则用常数1表示
        • 只保留时间函数中的最高阶项
        • 如果最高阶项存在，则省去最高阶项前面的系数

      T(n) = 3n => T(n)=O(n)

      T(n) = 5logn => T(n) =O(logn)

      T(n) = 2 => T(n) =O(1)

      T(n) = 0.5n 2 + 0.5n => T(n) =O(n 2 )

  1. 空间复杂度(对一个算法在运行过程中临时占用存储空间大小的量度)

    • 双重循环VS散列表

    • 常见的空间复杂度（S(n)=O(f(n))）

      • 常量空间

        S(n)=O(1)

      • 线性空间

        S(n)=O(n)

      • 二维空间

        S(n)=O(n^2)

      • 递归空间

        S(n)=O(n)

        (n：递归的深度)
